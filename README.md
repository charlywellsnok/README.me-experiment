# 🌊 Sea Monster Crowdfunding

A simple front-end web application that displays and analyzes crowdfunding data for independent games. Built with vanilla JavaScript, this project demonstrates DOM manipulation, data handling, and interactive filtering.

---

## 📌 Overview

Sea Monster Crowdfunding is a browser-based app that allows users to explore a collection of crowdfunded games. It provides insights into funding performance and lets users filter projects based on their funding status.

---

## 🚀 Features

### 🎮 Game Listings
- Dynamically displays game cards
- Each card includes:
  - Image
  - Name
  - Description
  - Number of backers

### 📊 Statistics Dashboard
- Total contributions (backers)
- Total amount raised
- Total number of games

### 🏆 Top Funded Games
- 🥇 Highest funded game
- 🥈 Second highest funded game

### 🔍 Filtering Options
- Show unfunded games
- Show funded games
- Show all games

### 📝 Dynamic Summary
- Displays a real-time summary of funding data
- Adjusts grammar based on number of unfunded games
